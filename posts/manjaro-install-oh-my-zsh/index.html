切换终端

Manjaro 默认集成了 ZSH，切换一下终端就 OK 了。

1
2
3
4
5
6

# 查看当前已安装的shell种类
$ cat /etc/shells
# 切换当前用户
$ chsh -s /bin/zsh
# 修改文件切换 root 用户
$ sudo vim /etc/passwd

安装程序

1
2

# 下载不了时，直接另存为 .sh 文件，然后执行 bash install.sh 。
$ sh -c "$(curl -fsSL https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"
