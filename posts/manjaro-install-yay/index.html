Manjaro 安装 Yay

Posted on 2020-05-12  by chishui

  • 程序安装
  • 配置 AUR 镜像
  • 基本命令
  • 参考文档

程序安装

1

$ sudo pacman -S yay

配置 AUR 镜像

1

$ yay --aururl "https://aur.tuna.tsinghua.edu.cn" --save

查看修改过的配置

1

$ yay -P -g

基本命令

用途命令
搜索$ yay -Ss package
安装$ yay -S package
卸载$ yay -R package
列表$ yay -Qs package
