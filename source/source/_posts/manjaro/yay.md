## 安装软件

```bash
$ sudo pacman -S yay
```

## 扩展管理

### 搜索

```bash
$ yay -Ss package
```

### 安装

```bash
$ yay -S package
```

### 卸载

```bash
$ yay -R package
```

### 查看列表

```bash
$ yay -Qs package
```
